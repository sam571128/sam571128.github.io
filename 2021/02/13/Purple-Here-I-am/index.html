寒假檢討以及成為紫牌了！

發表於 2021-02-13 Disqus：

好久沒有寫Blog 最近都一直富奸 XD

不過距離上一次也發生了不少事情 就來細數一下這個寒假到現在有哪些令人印象深刻的事情吧

IOICamp

這個寒假最愉快的莫過於 IOICamp 了

在那裡見到了各位電神，跟上一次去 IONCamp 時那種誰都不認識的情況不一樣了

這次見到了 Code Community 的人 學到的東西也不少

然後明明住在臺北還填住宿 從臺大到西門町還比到我家遠 （還被爸媽說幹嘛不回家睡 (ﾉ>ω<)ﾉ

每天的比賽也讓我見識到了自己實力的不足

第一天

一開始就拿到一本 IOICamp 講義 （為了這本來的

第一天的早上是電仁的喵喵時間

他分享了許多他以前的故事

還有他如何成為國手

還滿有趣的

中午則是和 benson0402 決鬥

而下午則是圖論

圖論課倒是沒學到什麼東西

沒啦 只是講師有很大的比重放在比較簡單的內容

跳過了圖的連通性 直接去講樹的演算法

不過樹論在 IONCamp 已經學到不少了 所以有很多這次就沒學到太多新的

新奇的東西大概就是 虛樹 了

然後講師還特別講的很快 因為他說隔天的個人賽會出現一題二分圖最大匹配 Σ(*ﾟдﾟﾉ)ﾉ

然後晚上就是 趣味賽 了

整場被隊友 Carry Orz

然後晚上則是走好久到飯店

晚上雞塊偷跑出去 還被波神列入觀察名單

第二天

早上第一堂課就是數學

雖然自己在學校範圍內的數學都是非常強的

但是競賽的數學跟學校內那種偏計算的還是不太一樣

一開始就是教 排列組合

不過我排列組合本來就不算太好 （？

所以其實也算是學到了一些東西 （然後 Foxyy 和 penguin71630 一直討論卡特蘭數 Orz

下午則是進階資結 波神的課真的讚

不過李超線段樹之前自己學過了

比較新奇的大概是 IOI 2018 Seats 還有暴力線段樹吧

然後終於能理解之前一直在 CF 看到的 Segment tree beats 了

晚上個人賽意外的拿到了第8名 應該是要很開心的

可是一題莫隊假解 一題亂猜二分圖最大匹配（畢竟講師前一天強調）

之後的比賽可不能靠假解上分呢 ｡ﾟヽ(ﾟ´Д`)ﾉﾟ｡

第三天

第三天的一開始就是字串

照理來說我應該是沒學過太多字串

但發現幾乎都聽過（除了 AC 自動機

所以前半就去記憶這些演算法 （然後打成模板

後半就去聽 AC 自動機

然後 Suffix Array 因為自己在 CF EDU 學過了 還寫過一篇 Blog

坐我旁邊的隊友聽不懂 Suffix Array 所以我就丟了我的 Blog 給他看 d(`･∀･)b

下午教的基礎 DP + DP 優化 （？ 是不是哪裡怪怪的

反正前半就睡下去了 後半才開始聽

但是除了 斜率優化 和 Aliens Trick 以外都聽不懂 QQ

晚上的團體賽也是靠著隊友拿分 Orz

第四天

第四天學的東西照理來說對於高中競程選手來說是不必要的

分別是 網路流 (Network Flow) 以及 賽局理論 (Game Theory)

但是網路流的演算法之前因為戳過 Flow 的題目

去學了起來 所以早上倒是都聽得懂 (ゝ∀･)

但下午除了 Nim Game + Sprague-Grundy Theorem 以外 都聽不太懂

還講到有偏的 Hackenbush

晚上的個人賽整個大燒雞 除了賽局不會寫之外 Flow的裸題還一直 WA

但還是放個前幾名的記分板好ㄌ

第五天

正式賽來了 5個小時 ICPC制

一整個還是被隊友 Carry Orz

就連隊友青牌的想的都比我清楚 我整個毒瘤起來每題都想砸 flow QQ

但一整週下來真的很好玩

後來還跟 Gino 和 Wiwi 他們一行人去吃拉麵

IOICamp 回來之後

過了幾天之後 跟 Foxyy 和 Wiwi 約好了要一起打 PCCA 的練習賽

然後一早就跑去 Wiwi Carry 整場

Foxyy 在耍毒瘤亂想幾何題

不過還是很好玩

然後 CF gym 也可以 vir 這場

之後的兩個禮拜又照常打 CF (其實也才三場)

然後終於在昨天上了紫牌 (Candidate Master)

踏入競程後10個月 終於上了紫牌

接下來的目標就是進TOI選訓以及上黃牌了
